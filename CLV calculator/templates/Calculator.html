Klaviyo customer lifetime value (CLV) calculator

Identify your top customers

Use this tool to identify the customers who have spent the most with you over time. This tool considers both orders and returns/refunds.

  1. Add your email address to the Email address field below.
  2. Add your Klaviyo private API key to the Private API key field below. Learn how to find your private API key in Klaviyo
  3. Select a number between 1 and 100 to represent the size of your best customers list. For example, if you'd like to identify the top 10% of customers over all time, insert 10. If you'd like to identify the top 25% of customers, insert 25.
    4. Complete the form below, inputting your email, Klaviyo private API key, and the size of the group you'd like to identify.
  4. After several minutes, you'll receive an email indicating the CLV delimiting customers at or above the group size you selected. Use this number to create the following segment:
      Predictive analytics > Historic CLV > [number indicated in your email]
  5. Save your segment. This segment will contain your top X% of customers based on lifetime spend.
